## Goal
Replace the placeholder entities page with a real entity list (card grid) and a create dialog, so users can create and browse entities within a world.

## Purpose
This is the first user-facing entity interaction — without it, users can't add any worldbuilding content. The create dialog + card list makes the entity system tangible.

## Output
- Entity list page showing cards for the selected entity type
- Create entity dialog with name + description fields
- Empty state with create CTA when no entities exist
